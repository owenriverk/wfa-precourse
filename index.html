How to Use These Lessons

Work through modules in order. Each lesson takes 20–40 minutes. Complete all quizzes — they reinforce the most important concepts.

📚

Pre-Course vs. In-Class

These modules cover the cognitive knowledge. Skills practice (splinting, CPR, wound care) happens in class with hands-on guidance.

🔁

Review Anytime

You can revisit any module after the course as a reference. The glossaries and key points make great quick-reference tools on trips.

⚠️

Educational Use Only

These lessons support WFA training. Do not use as a substitute for hands-on training or professional medical care.
